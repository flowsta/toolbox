"GitHub is a brilliant mechanism for sharing code and collaborating on ideas. However, GitHub is terrible for journalists." ~ Chris Hutchinson

The Journalist Toolbox is a site to “give journalists easy and clear access to tools built for them".

Through community agreed standards for tools built for journalists, we can work together to create better tools that are easier to use across different newsrooms, and by publishers small and large around the world.

The Journalist Toolbox was a winning idea built at the GEN + BBC News Labs #newsHack event in June 2015. Our prize is a partnership with the BBC and Goldsmiths University to launch an initial version of the platform.

If you have direct questions about the project, we've written about the standards here, and the project itself here. Alternatively, tweet the creators on @chrishutchinson, @samjoiner, @stefioceccon and @kestermollahan.
